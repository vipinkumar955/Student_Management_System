# Student Management System

A **role-based Student Management System** with **Admin, Teacher, and Student roles** featuring secure authentication, REST API backend, and a user-friendly UI/UX interface.

---

## Features

- **Role-Based Access Control:**  
  - Admin: Manage teachers, students, and system settings.  
  - Teacher: Manage courses, assignments, and student performance.  
  - Student: View grades,  view assignments, and personal information.

- **Authentication & Security:**  
  - Secure login and registration  
  - JWT / token-based authentication  
  - Password hashing and session management

- **REST API Backend:**  
  - CRUD operations for students, teachers, and courses  
  - JSON-based API responses  
  - Easily extendable for future integrations

- **UI/UX:**  
  - Responsive and clean interface  
  - Easy navigation for all user roles  
  - User-friendly forms and dashboards

- **Tech Stack:**  
  - Backend: Django  
  - Frontend: React 
  - Database:  PostgreSQL 
  - Authentication: JWT
